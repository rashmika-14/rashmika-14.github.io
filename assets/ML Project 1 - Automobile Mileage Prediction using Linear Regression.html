#user defined function for outlier Treatment
 
def remove_outlier(d,c):
    #find q1 and q3
    q1=d[c].quantile(0.25)
    q3=d[c].quantile(0.75)
    #calculate iqr
    iqr=q3-q1
    #calculate upper bound (ub) and lower bound (lb)
    ub=q3+1.5*iqr
    lb=q1-1.5*iqr
    #select good data and exclude outliers
    good_data=d[(d[c]>lb) & (d[c]<ub)]
    return good_data

Multicoliner Variables:

  • When two independent (input) variables are highly correlated (i.e.Corr>0.8) then they are called multicolinear variables.
  • What happens if we have multicolinear variables?
  • Ans: Multicolinear variables impact the accuracy of prediction
  • What is the solution of multicolinearity?
  • Drop one of the multicolinear variables to resolve this issue.

Using Pearson correlation test we found that key drivers (input varibles) are
